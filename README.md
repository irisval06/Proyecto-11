# Proyecto-11
Priorización de hipótesis y análisis de experimento A/B
En este proyecto trabajé como analista de datos en una tienda online, con el objetivo de aumentar los ingresos mediante la priorización de iniciativas y la validación de cambios a través de experimentación controlada.

La primera fase consistió en la evaluación y priorización de hipótesis propuestas por el equipo de marketing. Para ello, apliqué los frameworks ICE (Impact, Confidence, Effort) y RICE (Reach, Impact, Confidence, Effort), lo que permitió identificar y comparar las iniciativas con mayor potencial de impacto. Este análisis evidenció cómo la inclusión del alcance (Reach) puede modificar significativamente la priorización estratégica.

En la segunda fase, analicé los resultados de un experimento A/B utilizando datos de comportamiento de usuarios y transacciones. Se realizó una limpieza y validación de los datos, incluyendo la detección de usuarios presentes en ambos grupos y la identificación de valores atípicos.

Posteriormente, desarrollé un análisis completo del experimento, evaluando métricas clave como ingresos acumulados, tamaño promedio de pedido y tasa de conversión. Se utilizaron visualizaciones para comparar el desempeño de los grupos a lo largo del tiempo y detectar tendencias relevantes.

Además, se aplicaron métodos estadísticos para probar la significancia de las diferencias entre los grupos, tanto con datos originales como filtrados para eliminar anomalías. Esto permitió obtener resultados más robustos y confiables.

Finalmente, se formuló una recomendación basada en evidencia sobre la continuidad o finalización del experimento, apoyando la toma de decisiones del negocio.

Este proyecto demuestra habilidades en priorización estratégica, análisis de experimentos A/B, tratamiento de datos atípicos, inferencia estadística y comunicación de resultados orientados a la optimización de productos digitales.
